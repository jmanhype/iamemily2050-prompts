# ⚙️ PROJECT MEMORY: PRISM FORGE
**Role:** AI Image Studio for Professional Creators
**Paradigm:** Visual Recipe Builder / High-End Optical Hardware (Think Leica OS, Capture One)

## [ANTI-SLOP DIRECTIVES: CRITICAL]
- **BANNED:** Glassmorphism, blurred translucent layers, noisy mesh gradients, glowing drop shadows, floating unanchored cards, playful/bouncy UI.
- **REQUIRED:** Brutalist minimalism, tactile hardware-inspired controls, 1px hairline structural borders (`rgba(255,255,255,0.08)`), harsh 0px clipping on edge-to-edge images, maximum data density.

## [DESIGN TOKENS]
- **Canvas:** Near-black Graphite (`#111111`). **Surfaces:** Deep Charcoal (`#1A1A1A`) and Elevated Slate (`#222225`).
- **Accent:** Ultraviolet (`#5D3FD3`) — STRICTLY reserved for active states, 'Final' actions, and micro-interactions. No decorative use.
- **Typography:** Headlines/UI: Clean Neo-grotesk (Semibold). Metadata/Seeds/Weights: Strict Tabular Monospace (e.g., JetBrains Mono).
- **Geometry:** 8pt spatial grid. Radii: 24px (Large Sheets), 18px (Cards), 14px (Controls). 

## [SCREEN ARCHITECTURE]
1. **Projects Home:** Grid of oversized hero thumbnails bleeding to edges. Metadata sits *outside* the image in a strict 2-column tabular format. Pinned 'Style Kits' at top in a tight horizontal scroll.
2. **Create Screen (The Viewfinder):** Top 60% is an edge-to-edge live preview well (zero padding). Bottom 40% is the "Composer Sheet" containing: Prompt field + 6 distinct Reference Slots (Subject, Style, Comp, Lighting, Material, Text). Slots use a stark, drag-to-fill "Visual Heat Bar" for weight, not numeric sliders.
3. **Compare Screen:** 2-up or 4-up grid with exactly 1px pure black gutters. Tapping a variant reveals a "Diff Inspector"—a dark overlay showing the prompt delta in monospace (like a GitHub code diff).
4. **Style DNA Builder:** System-level aesthetic editor. Looks like a professional color-grading suite mapping contrast curves and hex palettes to hard-edged saveable modules.
