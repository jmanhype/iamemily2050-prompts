Claude Fable 5 
It was fun making this music.

You are an elite orchestral composer, conductor, orchestrator, music theorist, and audiovisual director.

Create a completely original symphonic work that feels monumental, emotionally overwhelming, and timeless, without copying or quoting any existing composer, film score, melody, theme, or copyrighted work. Do not write “in the style of” any living artist. Instead, use general musical qualities: grandeur, tragic beauty, heroic ascent, sacred awe, storm-like percussion, luminous strings, noble brass, and celestial choir-like textures.

Goal:
Compose a breathtaking original orchestral piece that could stand beside legendary symphonic works in ambition, emotional architecture, orchestration, and memorability.

Deliver the project in five parts:

1. COMPOSITION CONCEPT
Give the title, emotional premise, story arc, duration, tempo map, key centers, major transitions, and overall dramatic structure.

2. ORCHESTRAL SCORE BLUEPRINT
Write a detailed orchestration plan for:
- Strings: violins I/II, violas, cellos, double basses
- Woodwinds: flutes, oboes, clarinets, bassoons
- Brass: horns, trumpets, trombones, tuba
- Percussion: timpani, bass drum, cymbals, tam-tam, snare, concert toms
- Color instruments: harp, celesta, piano, organ, optional wordless choir

Include motifs, counter-melodies, harmonic progressions, rhythmic cells, dynamics, articulations, and register choices.

3. MAIN THEMES
Create 3 original themes:
- Theme A: noble, unforgettable, heroic
- Theme B: tragic, lyrical, human
- Theme C: cosmic, mysterious, transcendent

For each theme, provide:
- solfege or note names
- rhythm values
- harmonic support
- suggested instrumentation
- emotional purpose
- where it returns later in transformed form

4. MACHINE-READABLE MUSIC OUTPUT
Generate one of the following:
A. MusicXML for notation software,
B. LilyPond score code,
C. Python code that generates a MIDI file,
D. Tone.js or WebAudio code for browser playback.

Choose the best option for creating a playable demo. Include enough detail for a 3–5 minute orchestral mockup. Make the result structured, clean, and executable.

5. SYNCHRONIZED VISUAL REPRESENTATION
Create a timecoded visual direction sheet for the music, with one visual cue every 10–15 seconds.

The visuals should feel like a living symphonic painting:
- cosmic architecture
- golden light
- storm clouds
- mountains
- cathedral-scale geometry
- constellations
- abstract waves responding to strings, brass, percussion, and choir textures

For each visual cue, include:
- timestamp
- musical event
- camera movement
- color palette
- motion intensity
- particle behavior
- lighting change
- symbolic imagery

Then provide a final prompt for an AI video generator or real-time visualizer that can animate the entire piece while the music plays.

Quality bar:
The result must feel original, serious, emotionally devastating, and technically coherent. Avoid generic “epic trailer music.” Use long-form symphonic development, thematic transformation, tension and release, orchestral contrast, silence, restraint, and a final cathartic climax.
